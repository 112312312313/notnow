1. Что это такое

ALBORD Launcher — это сторонний лаунчер Minecraft с открытым исходным кодом. Он не разработан и не одобрен компанией Mojang Studios или Microsoft.

Лаунчер скачивает игровые файлы напрямую с официальных серверов Mojang (launchermeta.mojang.com, resources.download.minecraft.net) — то есть с тех же URL, что и официальный лаунчер.

ИИ-проект: сам лаунчер ALBORD сделан ИИ на 100%. Перед использованием рекомендуется ознакомиться с исходным кодом и понимать ограничения pet-project версии.

2. Что нужно установить отдельно

  • Java — без неё Minecraft не запустится. Версия зависит от версии MC:
Версия MinecraftНужна Java
1.16 и старшеJava 8
1.17Java 16
1.18 – 1.20.4Java 17
1.20.5 и новееJava 21

Скачать Java можно бесплатно с adoptium.net. После установки укажите путь к java.exe в настройках лаунчера.

3. Что лаунчер умеет

  • Запускать любые релизы Minecraft (vanilla)
  • Snapshot, old_alpha и old_beta версии
  • Скачивать client.jar, libraries, assets, natives
  • Настраивать RAM (по умолчанию 2 GB)
  • Свои JVM/Game аргументы (для опытных)
  • Менять путь к Java и папке .minecraft
  • Сворачивать окно при запуске MC (экономия CPU/RAM)

4. Что лаунчер НЕ умеет

  • ❌ Forge / Fabric / Quilt — только vanilla
  • ❌ Менеджер модов и модпаков
  • ❌ Официальную авторизацию Microsoft / Mojang (только offline mode)
  • ❌ Управление скинами и плащами
  • ❌ Авто-обновление самого лаунчера

5. Где будут храниться файлы

По умолчанию лаунчер использует отдельную папку Minecraft. Вы можете изменить путь в настройках, чтобы не смешивать файлы с другими лаунчерами. 

Windows: %APPDATA%\.minecraft
macOS: ~/Library/Application Support/minecraft
Linux: ~/.minecraft

6. Про безопасность и проверку

Проект open-source: если вы не доверяете готовому файлу, скачайте исходники, проверьте код и соберите лаунчер самостоятельно. Это особенно важно, потому что ALBORD — pet-project.

  • Скачивайте лаунчер только с официальной страницы проекта.
  • Не вводите пароль от Microsoft/Mojang — лаунчер работает в offline mode.
  • Проверяйте файл антивирусом перед запуском.
